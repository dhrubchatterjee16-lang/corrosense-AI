CorroSense AI is a full-stack deep learning system for predicting metal corrosion rates in industrial environments. The neural network (6→128→128→64→32→1) is implemented from scratch in NumPy with Batch Normalization, Leaky ReLU, Adam optimizer, Huber loss, and dropout regularization — no TensorFlow or PyTorch. The app processes 5,000 real corrosion records across 63 material families, achieves a test MAE of 0.96 mm/year, and deploys on Streamlit with live training visualization, permutation feature importance, real-time severity prediction, and a corrosion control recommendation engine covering 10 materials and 8 environment types.
